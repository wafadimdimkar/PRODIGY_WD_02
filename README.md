# PRODIGY_WD_02
The Stopwatch Web Application is a simple yet functional time-tracking tool built using HTML, CSS, and JavaScript.
It provides an interactive interface where users can:<br>
Start the timer<br>
Pause the timer<br>
Reset the timer<br>
Record multiple lap times<br>
The project focuses on practicing DOM manipulation, time calculations, event handling, UI design, and JavaScript logic implementation.<br>
